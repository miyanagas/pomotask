# [PomoTask（ポモタス）](https://todo-app-2yl.pages.dev/)
## 概要
ポモドーロタイマーとタスク管理を融合したWebアプリケーションPomoTaskのリポジトリです

## 目的
1日のタスクプランをたててから、スムーズにタスクの実行に移れる環境を作るためのアプリケーションです

## 使い方
### 登録・ログイン
ログイン画面よりログインしてご利用ください<br>
アカウント作成がまだの場合は、新規登録ボタンから、遷移先の画面よりアカウントの新規作成をお願いします

<img width="720px" alt="新規登録" src="https://github.com/user-attachments/assets/83957d67-2c72-4741-966e-0c2a8fa383df">

### タスクの追加・閲覧
下記の画面がこのアプリのホーム画面です（ヘッダーのアプリ名よりどのページからもアクセスできるようになっています）<br>
新しいタスクの追加、作成したタスクの一覧を見ることができます

<img width="720px" alt="タスク管理画面" src="https://github.com/user-attachments/assets/9be5f592-76e1-4519-af67-8bc03281e217">

また、完了したタスクは下記のように真っ青で表示され、完了までにかかった時間が表示されるので、
どれくらいの時間かけたのかを確認することができます

<img width="720px" alt="タスク管理画面（２）" src="https://github.com/user-attachments/assets/d144fb4a-e182-437e-be88-ff0d7a1048b8">

### 作業用のポモドーロタイマー
作成した各タスクより、作業用のポモドーロタイマーを利用することができます<br>
ポモドーロタイマーはデフォルトでは、タスク25分、休憩5分のサイクルですが、自由に設定することができるため、
自分のペースに合わせた作業が可能です

<img width="720px" alt="ポモドーロタイマー" src="https://github.com/user-attachments/assets/c5c0da62-99c2-4085-8f96-bf5e8ccb0dbc">

また、集中力を上げるために、作業用のBGMを流すことも可能です<br>
Youtubeからお気に入りのBGMを流すことができます<br>
1つの動画だけですが、お気に入りとして保存することもでき、これでYoutubeを開いて作業する必要もありません

<img width="720px" alt="ポモドーロタイマーとBGM" src="https://github.com/user-attachments/assets/6c6b356d-4604-4519-9341-ccb173f78c2e">

### マイページ

アカウント情報の閲覧および編集はマイページより可能です（マイページはヘッダーの人型アイコンよりアクセスできます）

ログアウト・アカウントの削除もこちらからできます
